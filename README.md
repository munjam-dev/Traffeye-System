# 🚦 TraffEye - AI-Powered Smart Traffic Violation Detection & e-Challan System

## 🎯 Overview
A futuristic smart city traffic monitoring platform that detects violations using AI, generates digital challans, and provides a premium user experience with multi-language support.

## 🏗️ Architecture
```
TraffEye-System/
├── backend/              # FastAPI backend
├── frontend/             # Next.js frontend
├── ai-module/           # YOLOv8 + OpenCV + EasyOCR
├── database/            # MongoDB schemas
└── deployment/         # Docker & deployment configs
```

## 🧠 AI Detection Capabilities
- ✅ Helmet detection (present/absent)
- ✅ Triple riding detection (>2 riders)
- ✅ Signal violation detection
- ✅ Number plate recognition (EasyOCR)

## 🌍 Multi-Language Support
English, Hindi, Telugu, Tamil, Kannada, Marathi, Bengali, Gujarati, Malayalam, Punjabi

## 🎨 Design System
- Dark luxury theme with glassmorphism
- Futuristic smart city UI
- Real-time analytics dashboard
- Mobile-responsive design

## 🚀 Tech Stack
- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion
- **Backend**: FastAPI, Python
- **AI**: YOLOv8, OpenCV, EasyOCR
- **Database**: MongoDB Atlas
- **Maps**: Leaflet/Google Maps
- **Auth**: Google OAuth

## 📱 Key Features
- Real-time violation detection
- Digital challan generation
- User & admin dashboards
- Live analytics
- Payment redirection
- Mobile-friendly

## 🔧 Quick Start
```bash
# Clone repository
git clone <repository-url>
cd TraffEye-System

# Install dependencies
pip install -r backend/requirements.txt
npm install

# Start development servers
python backend/main.py
npm run dev
```
